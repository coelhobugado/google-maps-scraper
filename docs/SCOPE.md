# Escopo oficial

O produto é um aplicativo local para uma única pessoa por instalação. O núcleo inclui CLI, interface web local, SQLite, exportações, enriquecimento opcional e suporte PostgreSQL para processamento distribuído controlado.

Fora do escopo desta distribuição: gestão de contas de clientes, cobrança, planos, organizações, hospedagem pública turnkey e provisionamento automático de infraestrutura. Integrações externas devem permanecer opcionais e desacopladas do núcleo local.
